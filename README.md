# spek-gradle-kotlin [![ci-badge]][ci-travis]

Hello world for a [Spek], [Kotlin], and [Gradle] combination.

## Usage
1. [Install Gradle].
2. From bash on Linux, or PowerShell or Git Bash on Windows, type `gradle test` from the top level of this project.
